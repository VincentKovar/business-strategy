# The Eight Questions — Strategy Ledger A self-guided interview tool based on the "Eight Questions" GTM strategy framework. Walks a leader through eight questions one at a time, then calls the Claude API to generate three key insights and five ordered action items based on their answers. ## Usage Open `index.html` in a browser, or visit it via GitHub Pages once deployed (see below). No build step, no dependencies to install — it's a single self-contained HTML file (vanilla JS, no framework). ## Deployment (GitHub Pages) This folder is structured so GitHub Pages can serve it directly: 1. Push this repo to GitHub (see root-level instructions). 2. In the repo, go to **Settings → Pages**. 3. Under "Build and deployment," set **Source** to `Deploy from a branch`. 4. Set **Branch** to `main` (or whichever branch you push to) and folder to `/ (root)`. 5. Save. GitHub will publish the whole repo; this file will be live at: ``` https://.github.io/business-strategy/8-questions/ ``` (GitHub Pages automatically serves `index.html` when a folder is requested, which is why this file is named `index.html` rather than something else.) It can take 1–2 minutes for Pages to build after each push. ## Notes - No data is saved or persisted — refreshing resets the tool. - The analysis step calls `api.anthropic.com` directly from the browser; it will only work where that endpoint is reachable and authorized for your use. - Export options (Save as PDF, Save as Markdown) are available on the results screen.
